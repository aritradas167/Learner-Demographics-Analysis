# EduPro Learner Demographics and Course Enrollment Analysis

This project analyzes learner demographics and course enrollment behavior on the EduPro platform using Python and Streamlit.

## Features
- KPI Dashboard
- Age Group Analysis
- Gender Analysis
- Course Category Analysis
- Course Level Analysis
- Heatmap Visualization
- Downloadable Filtered Data

## Tools Used
- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn
